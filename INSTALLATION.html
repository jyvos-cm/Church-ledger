Guide d'Installation PWA — Fonctionne Hors Ligne

✅ Bonne nouvelle ! Cette application fonctionne comme une vraie APK Android — elle s'installe sur votre écran d'accueil et fonctionne 100% sans connexion internet.

📱 INSTALLATION SUR ANDROID

Étape 1

Héberger les fichiers

Copiez le dossier church-ledger-pwa/ sur un hébergeur web. Options gratuites :

  • GitHub Pages — github.com → nouveau dépôt → activer Pages
  • Netlify — netlify.com → glisser-déposer le dossier
  • Vercel — vercel.com → importer le dossier
  • Serveur local — ex: python3 -m http.server 8080
Étape 2

Ouvrir dans Chrome Android

Sur votre téléphone Android, ouvrez Google Chrome et accédez à l'URL de votre site.

  • Chrome doit être utilisé (pas Firefox, pas Samsung Internet)
  • La première ouverture nécessite internet pour télécharger l'app
Étape 3

Installer sur l'écran d'accueil

Une bannière dorée apparaîtra en bas : «Installer Church Ledger»

  • Appuyez sur «Installer»
  • Ou : Menu Chrome (⋮) → «Ajouter à l'écran d'accueil»
  • Confirmez l'installation
Étape 4

✅ Prêt ! Fonctionne hors ligne

L'icône Church Ledger apparaît sur votre écran d'accueil. Appuyez dessus — l'app s'ouvre en plein écran, sans barre de navigation Chrome, comme une vraie application Android.

  • ✅ Toutes les données sont sauvegardées localement
  • ✅ Fonctionne sans connexion internet
  • ✅ Se lance instantanément

🍎 INSTALLATION SUR iPhone/iPad

Safari requis

Utiliser Safari sur iOS

  • Ouvrir l'URL dans Safari (obligatoire sur iOS)
  • Appuyer sur l'icône Partager (carré avec flèche ↑)
  • Choisir «Sur l'écran d'accueil»
  • Confirmer → l'icône Church Ledger apparaît

💻 UTILISATION SUR ORDINATEUR

Chrome / Edge

Installation sur PC/Mac

  • Dans Chrome ou Edge, une icône d'installation (⊕) apparaît dans la barre d'adresse
  • Cliquez dessus → «Installer»
  • L'app s'ouvre dans sa propre fenêtre sans navigateur
⚠️ Important : Les données sont stockées localement sur chaque appareil (localStorage). Pour partager les données entre plusieurs appareils, il faudra ajouter une base de données en ligne.
